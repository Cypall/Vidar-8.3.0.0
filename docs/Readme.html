2. How to Use

  1. Obtain the latest version of Vidar EMU.
  2. If you have a previous configuration for Vidar. BACK UP YOUR FILES. I cannot stress how important this is.
    Note that some files may cause duplicates of NPCs in older vidar installs. Conflicting NPCs are often due to my removing of some NPCs or replacing them with alternate script files, however, the installer WILL remove NPCs that are no longer needed.
  3. Run VCP-Install-xx-x.exe and follow the instructions.
  4. Select the components you wish to install - all are selected by default. Highlight a component to see a brief description of what it will do.
    Note: the installer will overwrite any and ALL files with newer versions of the file. Once again, before proceeding you should back up your files.
  5. Edit vidar.ini to your hearts content.
  6. Run, and hopefully enjoy.
