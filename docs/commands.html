Command Table: Vidar Commands:

Public Commands:

Command

Description/Usage
#wmessage <on of off> Turns the welcome message on and off.

Game Master Commands:

Command

Description/Usage/ID
#save Saves position
ID: SAVERETURN
#return Return to save position
ID: SAVERETURN
#die Kill yourself
ID: KILLDIEALIVE
#alive Revive yourself
ID: KILLDIEALIVE
#wmessage <on or off> Turns the welcome message on and off.
#hcolor <0-13> Changes hair colour.
ID: CHANGECOLORSTYLE
#ccolor <0-77> Changes clothing colour. Note: To exceed normal dye colours, your client must support them.
ID: CHANGECOLORSTYLE
#hstyle <0-19> Changes hair style.
ID: CHANGECOLORSTYLE
#kill <player name> Kills player_name.
Player name is case sensitive
ID: KILLDIEALIVE
#goto <player name> Warps to that player.
Player name is case sensitive
ID: GOTOSUMMONBANISH
#summon <player name> Warps that player to your location.
Player name is case sensitive
ID: GOTOSUMMONBANISH
#warp <map> <x> <y> Warps your player.
Map is case sensitive and must be lowercase.
Player name is case sensitive
Usage: #warp prontera 150 190
ID: WARP
#teleport <map> Randomly teleport to the specified map.
Map is case sensitive and must be lowercase.
ID: WARP
#silence <playername> <time> Mutes player's skills and talking. Time is in seconds.
Usage: #silence Lone 60 - will disable chatting and skills for 60 seconds.
#banish <player name> <map> <x> <y> Warps that player to location given.
Player name is case sensitive
Map is case sensitive and must be lowercase.
Usage: #banish Lone prontera 150 190
ID: GOTOSUMMONBANISH
#kick <player> Kicks <player>.
Player name is case sensitive
ID: KILLDIEALIVE
#kickall Kicks all players.
ID: KILLDIEALIVE
#ban <player> Bans <player> from server
Player name is case sensitive
ID: KILLDIEALIVE
#weather <option> Play weather effect.
Effects are: rain, snow, blossom, leaves, fireworks, rainbow, fog
ID: CHANGEWORLD
#effect <1-491> Play effect animation.
Requires full GM privileges
#gvg [on/off] Turn GVG on or off.
ID: CHANGEWORLD
#pvp [on/off] Turn PVP on or off.
ID: CHANGEWORLD
#refine <1-10> Refine equipment
You must relog or warp before the changes will be seen.
Using #goto Your_Name will work as well.
ID: REFINE
#job <1 - 66> Changes job.
ID: CHANGEJOB
#blevel <1-99[999]> Changes base level. Can go up to 999.
ID: CHANGELEVEL
#jlevel <1-50[70]> Change job level
ID: CHANGELEVEL
#changestat <0-5> <0-99> Changes character stat
0 = STR, 1 = AGI, etc.
ID: CHANGESTATSKILL
#skillpoint <1-1000> Changes your skill points
ID: CHANGESTATSKILL
#skillall Gives max level for all skills in your class
Also gives all class specific quest skills.
ID: CHANGESTATSKILL
#statall Changes all stats to 99 and adds 1000 stat points
ID: CHANGESTATSKILL
#zeny <1-1000000> Adds that amount to your current total. Adding too much can cause negative amounts.
#changeskill <skill id> <level> Changes level of specified skill.
Skill IDs can be found in the skill_db.txt.
#changeskill 28 10 - Gain level 10 heal in skill list.
ID: CHANGESTATSKILL
#item <item number> <quantity> Create item using item ID, for an alternative command, see /item.
Item digits can be found in the item databases.
Usage: #item 501 1 - Get 1 red potion.
ID: ITEMSUMMON
#monster <MONSTER_NAME> <Quantity[1-20]> Spawn <quantity> of MONSTER, for an alternative command, see /monster.
Monster names can be found in the mob databases
Usage: #monster PORING 1
ID: MONSTERSUMMON
#speed <20-200> Changes your walking speed. [Default is 150]
ID: CHANGEOPTION
#whois Prints what users are online to the chat window.
ID: WHOIS
#option <option> Changes specified options. Peko can ONLY be used as KNIGHT or CRUSADER.
Options are: peko, falcon, sight, cart, off
ID: CHANGEOPTION
#stat # # # # Show status infliction/Options
Examples:
#stat 0 0 32 0 - Pecopeco (Knight and Crusader only)
#stat 0 1 0 0 - Poisoned
#stat 1 0 0 0 - Stone Curse
ID: CHANGEOPTION
#raw Send raw packet to the server
ID: RAWUNIT
#unit usage: #unit [hex value/integer] - Display a map unit animation. (Like a square from 'Sanctuary')
ID: RAWUNIT

Aegis Commands:

Command

Description
/b <Text> Normal broadcast.
ID: BROADCAST
/nb <Text> Anonymous broadcast.
ID: BROADCAST
/bb <Text> Blue broadcast. Tricky to use at first.
Three extra characters before the actual text you want to send are needed
Usage: /bb 000 Hello
ID: BROADCAST
/monster <MONSTER_NAME> Spawn monster.
Usage: /monster PORING - Spawn one poring.
ID: MONSTERSUMMON
/item <Item_Name> Create item.
Usage: /item Red_Potion - Get 30 red potion each, equipment will only give 1 each.
ID: ITEMSUMMON
/hide GM Hide.
/resetstate Reset stat points.
ID: CHANGESTATSKILL
/resetskill Reset skill points.
ID: CHANGESTATSKILL
